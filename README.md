# README

## Описание

Это простая Bash-утилита для отображения структуры каталогов в виде дерева. Она позволяет просматривать содержимое указанной директории, а при необходимости — включать в вывод и файлы.

## Установка
- Скопируйте репозиторйи себе локально
- Для удобства рекомендуется установить путь к директории в которой находится скрипт в переменную среды PATH

## Использование

```bash
./tree[-f] [путь]
```
### Параметры

- `-f` — (опционально) показать не только каталоги, но и файлы.
- `[путь]` — (опционально) путь к директории, структуру которой нужно вывести. По умолчанию используется текущая директория (`.`).

### Примеры

1. Вывести дерево текущей директории (только каталоги):
   ```bash
   ./tree
   ```

2. Вывести дерево указанной директории с файлами:
   ```bash
   ./tree -f /home/user/projects
   ```

3. Вывести дерево директории без файлов:
   ```bash
   ./tree /var/log
   ```

## Особенности

- Выводится наглядное дерево с отступами и символами `├──`, `└──`, `│`.
- Поддерживается сортировка элементов по алфавиту.
- Если указан несуществующий или недиректорный путь, скрипт сообщит об ошибке.

## Требования

- Bash (должен быть установлен по умолчанию в большинстве Linux-дистрибутивов).
